从对话到上线 · From Context to Launch

一个关于「非精深开发者如何通过 AI Agent 完成个人网站设计与上线」的实验性项目。

项目简介

本仓库记录了：
- 一个 AI Native 的个人网站构建流程
- 一个非开发者与 Agent 协作的真实案例
- 一套给 AI Agent 使用的网页生成 skill
- 一份给新手人类阅读的建站白皮书

项目核心不是“写代码”，而是：
> 如何通过 Context + Agent 协作，
> 让普通人获得过去只有专业开发团队才能拥有的生产能力。
